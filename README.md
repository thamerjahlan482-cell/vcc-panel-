# 🎮 Vibe City - FiveM Website Template

موقع احترافي جاهز للبيع لسيرفرات FiveM مع لوحة تحكم كاملة

---

## 🚀 تشغيل المشروع

### المتطلبات
- Node.js v18+
- npm

### خطوات التشغيل

```bash
# 1. فك ضغط المجلد وادخله
cd vibecity

# 2. تثبيت المكتبات
npm install

# 3. تعديل إعدادات الدخول (مهم!)
# افتح ملف .env وغيّر:
# ADMIN_USERNAME=admin
# ADMIN_PASSWORD=admin123

# 4. تشغيل السيرفر
npm start

# 5. افتح المتصفح
# الموقع: http://localhost:3000
# لوحة التحكم: http://localhost:3000/admin
```

│   └── admin/         ← صفحات لوحة التحكم

## 🎛️ لوحة التحكم

### الدخول
- الرابط: `/admin`
- المستخدم: `admin` (قابل للتغيير في .env)
- كلمة المرور: `admin123` (غيّرها فوراً!)

### الصلاحيات
- ✅ تعديل اسم السيرفر والشعار
- ✅ تغيير الألوان والوصف
- ✅ تعديل روابط التواصل الاجتماعي
- ✅ تعديل الإحصائيات
- ✅ إضافة/حذف/إخفاء صناع المحتوى
- ✅ إضافة/تعديل/حذف أقسام القوانين

---

## 🌐 الرفع على Render.com (مجاني)

1. ارفع المشروع على GitHub
2. روح على render.com وأنشئ حساب
3. New → Web Service → اختر الـ repo
4. Build Command: `npm install`
5. Start Command: `npm start`
6. أضف المتغيرات من ملف .env
7. Deploy!
